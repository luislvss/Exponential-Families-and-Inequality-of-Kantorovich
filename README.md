# Exponential Families and Inequality of Kantorovich

## 📌 Overview
This repository contains a comprehensive statistical analysis framework focusing on Exponential Families of Distributions and the Kantorovich Inequality. The project was developed as part of my advanced statistics research at the Faculty of Mathematics and Computer Science (University of Bucharest).

## 🚀 Technical Highlights
* **Extensive Codebase:** Over 3,600 lines of R code written in RMarkdown.
* **Stochastic Modeling:** Implementation of Monte Carlo simulations and advanced random variable generation (Inverse Transform, Rejection Sampling).
* **Theoretical Verification:** Numerical verification of the Central Limit Theorem (CLT), Law of Large Numbers (LLN), and estimator properties (consistency, unbiasedness).
* **Data Visualization:** Advanced 3D visualizations for probability densities and variable transformations.

## 📁 Files Included
* `cod.Rmd`: The main RMarkdown source code containing all algorithms, simulations, and statistical tests.
* `Proiect Statistică.pdf`: The final compiled academic report featuring mathematical proofs and generated plots.

## 🛠️ Technologies & Tools
* **Languages:** R, RMarkdown
* **Concepts:** Probability Theory, Monte Carlo Simulations, Quantitative Analysis
